# 🤖 Gemini AI Chatbot

An AI-powered chatbot built using **Google Gemini 2.5 Flash API**, **Python (Flask)**, **HTML**, **CSS**, and **JavaScript**. The application provides real-time conversational responses through a clean and responsive web interface.

---

## 📌 Features

- 💬 Real-time AI conversations
- ⚡ Powered by Google Gemini 2.5 Flash
- 🌐 Responsive web interface
- 🔥 Flask backend
- 🎨 Modern HTML/CSS UI
- 📡 JavaScript Fetch API integration
- 📝 Easy to customize
- 🚀 Beginner-friendly project structure

---

## 🛠️ Technologies Used

- Python
- Flask
- Google GenAI SDK
- HTML5
- CSS3
- JavaScript
- Gemini 2.5 Flash API

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/gemini-ai-chatbot.git
```

```bash
cd gemini-ai-chatbot
```

---

### Create Virtual Environment

Windows

```bash
python -m venv venv
venv\Scripts\activate
```

Linux/Mac

```bash
python3 -m venv venv
source venv/bin/activate
```

---

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Configure API Key

Create a `.env` file in the project root.

```env
GEMINI_API_KEY=YOUR_API_KEY
```

> Never expose your API key in source code or public repositories.

---

## ▶️ Run the Application

```bash
python app.py
```

Open your browser and visit

```
http://127.0.0.1:5000
```

## 📁 Main Files

### app.py

- Flask server
- Handles API requests
- Connects frontend and backend

### chatbot.py

- Initializes Google GenAI client
- Sends prompts to Gemini
- Returns AI responses

### index.html

- Chat interface
- User input
- Message display

### style.css

- Responsive design
- Chat bubbles
- Layout styling

### script.js

- Sends messages using Fetch API
- Updates chat dynamically
- Displays AI responses
